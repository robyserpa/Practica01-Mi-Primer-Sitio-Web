DESARROLLO
Se creo 9 páginas y la principal con la extensión index.html y tomando en consideración la estructura propuesta para todas ellas.

Estructura index.html
Head 
Esta página está enfocada a información sobre el sistema operativo Android, tiene como title Información Android, utilizo <meta charset="utf-8" /> para especificar que se va a usar caracteres especiales.
Utilizo una imagen como logo principal <img src="images/logo_android.jpg" alt=""/> para todas las páginas, el cual también se lo utiliza para redirigirse a la página principal que sería index.html 
<a href="index.html"><img src="images/logo_android.jpg" alt=""></a> se lo utiliza en las demás páginas.
 Header 
Para la navegación utilizo una lista desordenada con sus respectivos vínculos utilizando la etiqueta <a>, obviando la pagina en la que se encuentre, se actualiza la lista.
En el header también se incluye el titulo de la página y una imagen relacionada al tema, esta página se titula Información Android. 
 Section, Article, Aside
Tiene su respectivo titulo y dentro cuenta con 1 article con una imagen que tiene dimensiones específicas <img src="images/interfaz_android.jpg" alt="" height="400" width="700"> se usa el height para el largo de la imagen en pixeles y el width para el ancho. Uso de una etiqueta h2 para el titulo relacionada al artículo, el cual cuenta con 3 aside y este contienen información del artículo, utilizo la etiqueta <h3> para los títulos de aside y <p> para los párrafos de información. 
 Section 
La otra sección esta reapartida con la misma estructura que la anterior y en el segundo aside del article 
Se encuentra la tabla que fue generada en https://www.tablesgenerator.com/ en formato html.
 Footer
Contiene información: Nombre, correo utilizando el mailto, teléfono utilizando tel, y un mensaje de derechos reservados con el símbolo ©
El pie de la página está estructurado de la siguiente manera.
